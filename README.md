gnome-terminal-quick-launch
===========================

A little script to call Gnome Terminal like Guake, Yakuake (without the dropdown effect)

----------

### Dependencies:
  - wmctrl
  - xdotool

----------

### How to use:
  - copy `bin/open-terminal.sh` to whatever you want
  - add a keybind to `open-terminal.sh`
  - be happy
